HTML Elements Showcase – Inline vs Block

HTML elements are divided into block and inline types to organize how content appears on a webpage. Block elements, such as <div>, <p>, or headings, always start on a new line and stretch across the full width, making them useful for structuring the main layout. Inline elements, like <span>, <a>, or <strong>, do not break the text flow; instead, they only occupy as much space as their content needs, making them ideal for styling or linking within a line of text. This division helps developers control both the overall structure and the finer details of web content.

Block Elements

This is div tag

A generic container used to group other elements. It doesn’t add meaning but is super useful for layout and styling with CSS.


This is Paragraph tag

Defines a paragraph. Always starts on a new line and creates space before and after the text.


This is a section tag

Defines a section of a webpage, like a chapter or topic group. Helps in dividing content meaningfully.


This is aside tag

For side content, like ads, sidebars, or extra notes.


This is an article tag

Used for self-contained content like a blog post, news story, or forum entry.


This is header tag

Defines the header area of a webpage or section (usually contains logo, nav, title).


This is a navigation tag

Specifically for navigation links.


This is table

Used to create tables; the entire table is block-level.


This is a block quote

For longer quotations, usually indented.


This is an address tag

Represents contact information (like an address, email, or phone number).


Inline Elements

This is spantag.

Span is an inline container with no new line.


This is an anchor tag.

Hyperlink (used to link to other pages or sections).


This is Strong tag.

Makes text bold and gives it importance.


This is Bold tag

Bold text (without semantic importance like strong).


This is em tag.

Emphasizes text, usually shown in italics.


This is italic tag.

Italic text (without semantic importance like em).


This is underline tag.

This is use to underlines text.


This is an abbr tag

Represents an abbreviation, can show a tooltip on hover with title.


This is cite tag.

Used to reference a title of a work (like a book, movie, article).


This is mark tag.

Highlights text with a yellow background by default.


This whole line is in quote tag.

It is use to write short inline quotation.
